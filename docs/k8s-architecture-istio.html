Three Gateway Options Compared

Nginx Ingress

  • Retiring March 2026
  • Simple, battle-tested
  • No sidecar / no mesh
  • Annotation-heavy config
  • No mTLS between pods
  • Best for: legacy / simple

Envoy Gateway

  • Gateway API (GA, CNCF)
  • Envoy data plane only
  • No sidecar injection
  • Lightweight, focused
  • WASM extensible
  • Best for: Sigma (Envoy stack)

Istio + Gateway

  • Gateway API + full mesh
  • Sidecar auto-inject
  • mTLS, tracing, Kiali
  • Heavier (istiod + sidecars)
  • Canary / traffic splitting
  • Best for: large fleet / zero-trust
