Nginx Ingress vs Envoy Gateway

Nginx Ingress (retiring March 2026)

apiVersion: networking.k8s.io/v1
kind: Ingress
metadata:
  name: sigma-ingress
  annotations:
    nginx.ingress.kubernetes.io/...
spec:
  rules:
  - host: sigma.yourdomain.com
    http:
      paths:
      - path: /api
        backend:
          service:
            name: sigma-api-service
            port: { number: 3000 }

Envoy Gateway (Gateway API GA)

apiVersion: gateway.networking.k8s.io/v1
kind: Gateway
metadata:
  name: sigma-gateway
spec:
  gatewayClassName: eg
  listeners:
  - name: https
    port: 443
    protocol: HTTPS
---
apiVersion: gateway.networking.k8s.io/v1
kind: HTTPRoute
metadata:
  name: sigma-api
spec:
  parentRefs:
  - name: sigma-gateway
  rules:
  - matches:
    - path: { type: PathPrefix, value: /api }
    backendRefs:
    - name: sigma-api-service
      port: 3000
